# PETIQUE | 분양 게시판

PETIQUE는 반려동물 정보를 기록하고 커뮤니티와 분양 기능을 함께 다루는 팀 프로젝트입니다.

이 프로젝트에서는 분양 게시판 쪽을 맡았습니다. 겉으로는 게시글 기능처럼 보이지만 실제로는 신청, 승인, 완료, 후기까지 상태가 이어지는 파트라서, 게시판보다 프로세스에 가깝게 보고 정리한 작업이었습니다.

---

## 프로젝트 소개

- 반려동물 정보와 커뮤니티 기능을 함께 다루는 서비스
- 분양 게시글, 신청 내역, 알림, 후기까지 이어지는 구조
- 작성자 / 신청자 / 방문자에 따라 다른 상태와 동선이 필요한 기능

---

## 담당 역할

- 분양 리스트 / 상세 / 신청자 관리 화면 구현
- 분양 신청 / 취소 / 승인 / 거절 / 완료 프로세스 설계 및 구현
- 상태별 버튼 노출과 권한 처리
- 분양 후기, 알림, 마이페이지 동선 연결
- 분양 관련 테이블 연동 로직 구현

---

## 사용 기술

- Spring MVC
- JSP, JSTL
- JdbcTemplate
- Oracle
- Maven, Git

---

## 구현하면서 신경 쓴 점

- 분양을 일반 게시판이 아니라 상태가 바뀌는 기능으로 보고 정리했습니다.
- 리스트와 상세가 같은 상태 기준을 보여주도록 맞췄습니다.
- 승인과 완료는 화면에서만 막지 않고 서버에서 한 번 더 검증하도록 두었습니다.
- 완료 뒤 후기 작성과 알림까지 이어져 사용자가 다음 행동을 자연스럽게 찾을 수 있게 붙였습니다.
